== Description == 


Plugin allow you to add facebook like/ Recommend button to your post/pages/archive or home page in few clicks.


== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `add-facebook-like-button` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
